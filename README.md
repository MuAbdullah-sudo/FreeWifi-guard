# 🛡️ Free-Wi-Fi Guard
**An Automated Security Scanner for Open & Public Networks.**

## 📌 Project Overview
Free-Wi-Fi Guard is a Python-based security tool designed to protect users from "Evil Twin" attacks and malicious public hotspots. It scans the airspace for open Wi-Fi networks and analyzes their signal fingerprint to detect potential Man-in-the-Middle (MitM) threats.

## ✨ Features
- **Real-Time Scanning:** Monitors local airwaves for unencrypted (Open) Wi-Fi.
- **Signal Fingerprinting:** Analyzes BSSID and Signal Strength (dBm) to detect anomalies.
- **Instant Alerts:** Color-coded terminal alerts (Green for safe, Red for danger).
- **Lightweight:** Optimized for performance with minimal CPU usage.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **OS Support:** Linux (Monitor Mode required), Windows (Basic Scanning)

pip install -r requirements.txt
